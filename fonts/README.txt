NOTE / PROTECTION: Nordel is a proprietary typeface

A web font is always downloadable by the browser — it cannot be fully
hidden. We limit cross-domain reuse via CORS (see /_headers). Full
protection would require serving display text as SVG outlines and never
shipping the font file; revisit this if the body text moves to a free font.
